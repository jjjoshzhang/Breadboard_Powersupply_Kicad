# Breadboard_Powersupply_Kicad
Breadboard Power Supply (LM7805 + LM317) — KiCad Project

A compact, dual-output breadboard power supply module designed between June. 2025 – July. 2025 as a personal electronics project.
This board provides regulated 5 V and adjustable 3.3 V outputs using classic linear regulators (LM7805 and LM317), optimized for prototyping and educational use.
